AxonIQ Console Reference Guide

With AxonIQ Console, you can manage anything AxonIQ. Whether you want to use Axon Server, find performance bottlenecks in your Axon Framework application, or manage your event processors, AxonIQ Console has you covered.

For example, AxonIQ Console gives you deep insight into the performance of aggregates in your Axon Framework application, as you can see in the image below.

Features

AxonIQ Console provides a variety of features to help you manage your Axon Framework applications and Axon Server environments.

Axon Framework:

  • Use the many insightful metrics that tell the health of any part of the application.

  • Manage event processors of with ease. You can start, stop, and reset event processors, as well as scale them automatically.

  • Retry or delete items in Dead-Letter Queues without any hassle.

  • Visualize the message flow in your application.

  • Set up alerts to get notified when something goes wrong.

Axon Server:

  • Automatically cluster Axon Server instances.

  • Set up alerts for your Axon Server’s health

  • License files are automatically rotated for you

Many features are included in the Developer plan, free of charge. Check out the subscriptions page for more information.
