Clustering

This section addresses the specifics of clustering Axon Server nodes. It covers how to cluster nodes, what roles each node may have, as well as specific data Axon Server generates in a clustered scenario, with the config database and the replication logs sections.

The option to cluster Axon Server nodes is not supported in the Developer plan.
