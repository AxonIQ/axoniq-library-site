AxonIQ Playbook

Welcome to AxonIQ Playbook, a catalogue of patterns to support the AxonIQ community in modeling, implementing, and operating event-sourced systems.

Proper system design relies on a solid understanding of one’s business domain; AxonIQ Playbook embraces Event Modeling and its building blocks for this purpose, by documenting common problems and solutions related to system state changes and views, automations, integrations and more, along with code examples using Axon components.
