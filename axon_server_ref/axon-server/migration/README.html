Migration

There are two migration options that are available within Axon

Options Description

Standard Edition to Axon Server

Upgrading your Axon Server SE based deployment to an Axon Server EE based deployment

Non-Axon Server to Axon Server

Migrating your non-Axon Server based deployment (Relational DB) to an Axon Server based deployment (SE/EE)
Flow Control Non-Axon Server to Axon Server
