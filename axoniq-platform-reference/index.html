Axoniq Platform

Axoniq platform is the base of operations for Axon Framework and Axon Server. It’s designed to make your life easier with any Axon-based application. Whether you want to develop an Axon Framework 5 application using our AI Development agent, find performance bottlenecks in your Axon Framework application, or manage your event processors, Axoniq Platform has you covered.

Features

Axoniq Platform provides a variety of features to help you build and manage your Axon Framework applications and Axon Server environments.

Development Agent:

  • Use the Development Agent to build Axon Framework 5 applications with a prompt in a structured way.

  • Share and explore other user’s public projects.

Axon Framework:

  • Use the many insightful metrics that tell the health of any part of the application.

  • Manage event processors of with ease. You can start, stop, and reset event processors, as well as scale them automatically.

  • Retry or delete items in Dead-Letter Queues without any hassle.

  • Set up alerts to get notified when something goes wrong.

Axon Server:

  • Automatically cluster Axon Server instances.

  • Set up alerts for your Axon Server’s health

  • License files are automatically rotated for you

Many features are included in the Community plan, free of charge. Check out the subscriptions page for more information.
